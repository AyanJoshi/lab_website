About

Current Research Projects

Acoustic Side Channel Attack in Collaborative Robotics (Master's Thesis)

Designing an attack framework that exploits acoustic emissions from collaborative robot arms (Franka Emika Panda) to infer proprietary motion trajectories and operational states. Developing signal processing pipeline using spectral analysis and machine learning classifiers.

Inverse Mixed Reality (IMRAI) - NSF I-Corps

Developed AR application system (Unity/C#) for Magic Leap 2 to enable safe robot training by overlaying virtual hazards onto physical environments. Engineered UDP networking layer streaming motion-capture data at 90Hz into AR.

Robotic Fleet Security - "Phantom of the Warehouse"

Co-designed a hardware-level memory corruption attack targeting autonomous mobile robot (AMR) fleets. Executed KSM-enabled Rowhammer attack on ROS Parameter Server, inducing targeted bit-flips in kinematic configuration files.

SDR-Enabled Wireless Digital Twin (AFRL University Challenge)

Building a Spatiotemporal Wireless Digital Twin for mobile autonomous systems using USRP B210 SDRs and NVIDIA Jetson Orin. Implementing ML-driven channel prediction models integrated into ROS/Gazebo.
