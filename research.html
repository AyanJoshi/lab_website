Research

Our research focuses on Computing and Communications in various Internet-of-Things paradigms, with emphasis on autonomous systems, edge computing, and intelligent mobile systems.

Collaborative Perception over Vehicular Edge Computing

In smart transportation systems, local perception based on one vehicle's sensor data may not yield high accurate global perception. Multi-sensor fusion can greatly improve vehicular perception through collaborative computations with edge servers. We develop efficient task partitioning algorithms and real-time multi-sensor fusion at the edge to improve collaborative vision accuracy.

Collaborators: Samuel Thornton (UCSD), Yujen Ku (UCSD), Sujit Dey (UCSD)

Design Space Exploration for ML Applications

For complex computing-intensive AI tasks, system performance needs joint evaluation. We conduct design-space exploration for software-hardware co-design of application-driven adaptive computing systems, optimizing power, area, and speed while maintaining high accuracy and low latency.

Collaborators: Sujit Dey (UCSD), Anand Raghunathan (Purdue)

Resilient Computation for UAV Systems

The ability to execute complex signal processing and ML tasks in real-time is core to autonomy. We develop "Hydra", an architecture for flexible sensing-analysis-control pipelines over autonomous airborne systems, introducing the concept of information autonomy for edge-assisted UAV applications.

Collaborators: Marco Levorato (UCI)

Renewable Energy-driven Vehicular Edge Computing

We create machine learning models to predict small cell operation time based on communication traffic, computing demand, and environmental factors, maximizing renewable energy utilization in vehicular edge computing systems.

Robust UAV Communications

We design a robust multi-path communication framework for UAV systems, featuring a fully open-sourced integrated UAV-network simulator called "FlyNestSim" for testing and validation.

Software-Defined Edge Computing

We develop a content and computation-aware communication control framework based on SDN paradigm, using extended Berkeley Packet Filter (eBPF) for efficient multi-streaming in edge computing environments.

Cognitive Interference Control

We create a cognitive interference control framework for heterogeneous local access networks in urban IoT systems, optimizing transmission patterns for improved throughput and accuracy.
